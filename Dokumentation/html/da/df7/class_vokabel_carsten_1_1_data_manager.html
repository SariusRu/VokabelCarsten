VokabelCarsten.DataManager.DataManager ( )

DataManager-Constructor

This Constructor creates a new object of the type DataManager. After Creation the object contains a list of "unloaded" VocabBoxes. Unloaded Vocabboxes don't contain Vocabs, but a reference to the file where to find them. An Error is thrown if a reading-Error occurs.

bool VokabelCarsten.DataManager.readVocabBoxList ( )
private

Reads a list of Vocabs from a File

This methos is using the vocabFileList from the members to check if the file exists. If no File exists a file is created. After that the methos tries to read from the file using a FileStrean and a XML-Serializer. The results are cated to a list of VocabBoxes and then saved in vocabBoxes

Returns
False if something wen't wrong or the methos was interrupted. Otherwise always true.
